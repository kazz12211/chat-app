# chat-app

## 概要

Spring BootのWebSocketを使った簡単なチャットアプリケーション。

ソーシャルログイン機能を使ってチャットを開始します。
あらかじめソーシャルサービス（この場合はFacebookとGitHub）にアプリケーションの登録が必要です。
